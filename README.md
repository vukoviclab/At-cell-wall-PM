# At-cell-wall-PM
Codes and files for simulating plasma membrane and cell wall of A. thaliana leaf


The `parameters` directory contains the **CHARMM parameter files** specifically modified for various lipids used to build the *A. thaliana* plasma membrane (PM).

The `example` directory contains the example input files (`psf` and `pdb`) along with the `toppar` folder with the required parameters for the lipids.
